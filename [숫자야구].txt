[숫자 야구]
 조건 : 1~9까지 서로 다른 수로 이루어진 세자리 수
  같은 자리에 같은 숫자가 있으면 Strike
  같은 숫자가 다른 자리에 있으면 Ball
  같은 숫자가 전혀 없으면 Nothing

 예시) 컴퓨터의 숫자가 '123'일 경우, 내가 입력한 수가 '189'일 때
		-> 1 Strike
	컴퓨터의 숫자가 '123'일 경우, 내가 입력한 수가 '289'일 때
		-> 1 Ball
	컴퓨터의 숫자가 '123'일 경우, 내가 입력한 수가 '456'일 때
		-> Nothing
	컴퓨터의 숫자가 '123'일 경우, 내가 입력한 수가 '123'일 때
		-> Win
	컴퓨터의 숫자가 '123'일 경우, 내가 입력한 수가 '132'일 때
		-> 1 Strike, 2 Ball

요구 사항 : 
 1.들여쓰기 depth가 2를 넘지 않도록 (최대 depth는 1)
	-> 메서드를 분리하는 것이 요점
 2. else, switch, 삼항연산자 사용 불가
 3. 메서드는 한 가지 일만 하도록 구현(SOLID원칙)
 4. 핵심 로직을 구현하는 코드와 UI 로직을 분리
 5. commit 컨벤션 지키자
